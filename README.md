# 三甲医院数据看板展示大屏

## 项目简介

这是一个专为三甲医院设计的LED大屏数据看板展示系统，采用现代化的Web技术构建，支持多种视觉风格和宽屏显示布局。系统集成了医院运营监控、环境监控、停车场管理、网络监控等多个功能模块，为医院管理提供全面的数据可视化支持。

## 🎨 多风格支持

项目提供三种不同的视觉风格，满足不同场景和用户需求：

### 1. 经典风格 (Classic Style)
- **文件**: `index.html` + `styles.css` + `script.js`
- **特色**: 传统医院风格，四主题切换（浅蓝/浅绿/浅紫/浅粉）
- **适用**: 传统医院环境，注重稳定性和可读性

### 2. 莫兰迪风格 (Morandi Style)
- **文件**: `morandi-dashboard.html` + `morandi-styles.css` + `morandi-script.js`
- **特色**: 优雅柔和的莫兰迪色系，低饱和度设计
- **适用**: 高端医疗环境，注重视觉舒适度

### 3. 科技风格 (Tech Style)
- **文件**: `tech-dashboard.html` + `tech-styles.css` + `tech-script.js`
- **特色**: 冷调科技色系，现代化界面设计
- **适用**: 智能医院，注重科技感和未来感

## 功能特色

### 🏥 医院运营监控

- **核心运营指标**：收入情况、设备使用率、预约管理、医院负荷率、医护人员在岗情况
- **科室运营情况**：内科、外科、急诊科、儿科、妇产科、骨科、神经科、心血管科等8个科室实时状态
- **医疗质量指标**：院内感染率、再入院率、平均住院日、安全评分
- **患者流量趋势**：实时患者流量统计和趋势分析，支持日/月视图切换
- **收入分析**：门诊收入、住院收入、检查收入、其他收入占比分析
- **设备状态监控**：设备运行状态、维护中、故障、待机状态统计

### 📊 数据可视化

- **多种图表类型**：折线图、柱状图、饼图、仪表盘、雷达图、环形图
- **实时数据更新**：每5秒自动更新数据，每3秒更新监控数据
- **交互式图表**：基于ECharts 5.4.3的丰富交互体验
- **响应式设计**：适配1920px-2560px+的LED大屏显示
- **图表主题适配**：支持多主题颜色自动适配

### 🌡️ 环境监控

- **温度监控**：实时温度显示和仪表盘趋势图（0-40°C范围）
- **湿度监控**：湿度水平监控和预警（0-100%范围）
- **空气质量**：PM2.5监测和空气质量评级（优/良/中等级）
- **电力监控**：实时功耗监控和用电统计（800-1600kWh范围）

### 🚗 停车场管理

- **车位统计**：总车位245个、已使用191个、空闲54个
- **使用率监控**：78%实时停车场使用率统计
- **可视化展示**：饼图展示停车场使用情况，支持实时更新

### 🌐 网络监控

- **网络速度**：上行125Mbps、下行856Mbps实时监控
- **在线设备**：2,345个连接设备数量统计
- **网络延迟**：12ms网络延迟监控
- **流量趋势**：网络流量变化趋势图，支持上行/下行对比

- **状态指示器**：系统运行正常、部分科室繁忙、设备运行良好、库存充足等状态显示

## 技术架构

### 前端技术

- **HTML5**：语义化标签和现代HTML特性，支持全屏API
- **CSS3**：Flexbox、Grid布局、CSS变量、毛玻璃效果、渐变背景
- **JavaScript ES6+**：模块化编程、异步处理、类管理、事件处理
- **ECharts 5.4.3**：专业的数据可视化图表库，支持多种图表类型
- **Font Awesome 6.0.0**：图标库支持

### 核心特性

- **多风格支持**：三种不同的视觉风格，满足不同场景需求
- **响应式设计**：支持1920px-2560px+的LED大屏显示，多断点适配
- **主题切换**：经典风格支持四主题切换，其他风格支持明暗模式
- **数据模拟**：内置完整数据模拟器，支持实时数据更新和随机变化
- **全屏支持**：F11键全屏显示，专为大屏优化，支持全屏API
- **性能优化**：图表懒加载、数据缓存、动画优化、错误处理机制
- **实时更新**：多时间间隔数据更新（5秒、3秒、10秒、15秒）
- **状态管理**：系统状态实时监控和告警管理

## 使用方法

### 选择风格版本

1. **经典风格**：打开 `index.html` - 传统医院风格，四主题切换
2. **莫兰迪风格**：打开 `morandi-dashboard.html` - 优雅柔和风格
3. **科技风格**：打开 `tech-dashboard.html` - 现代科技风格

### 基本使用

1. **直接打开**：在浏览器中打开对应的HTML文件
2. **本地服务器**：建议使用本地HTTP服务器运行以获得最佳体验
3. **全屏显示**：按F11键进入全屏模式，适合LED大屏展示
4. **主题切换**：根据版本不同，支持不同的主题切换方式
5. **数据更新**：系统自动更新数据，无需手动操作
6. **响应式适配**：支持不同分辨率自动适配布局

### 拖拽布局功能

科技风格版本支持拖拽布局功能：
1. 点击右上角的布局设置按钮
2. 开启编辑模式
3. 拖拽模块到目标位置
4. 保存布局配置

## 部署建议

### LED大屏部署

- 推荐分辨率：1920x1080 或更高
- 浏览器：Chrome（推荐）或Firefox
- 全屏模式：使用F11键或浏览器全屏API
- 自动刷新：可设置浏览器自动刷新页面

### 服务器部署

- 静态文件服务器：Nginx、Apache等
- CDN加速：建议使用CDN加速图表库加载
- 缓存策略：设置适当的缓存头优化加载速度
